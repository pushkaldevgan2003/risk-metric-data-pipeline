# Risk Metrics Data Engineering Pipeline

## Overview
A production-style data engineering pipeline that ingests market data, applies data quality checks, computes financial risk metrics, and stores results in a relational database.

## Tech Stack
- Python
- Pandas, NumPy
- SQLAlchemy (2.x)
- MySQL
- Data Quality Checks

## Key Features
- End-to-end ETL pipeline
- Idempotent database loads
- Financial risk metrics (VaR, Volatility, Max Drawdown)
- SQL schema design
- Logging and error handling
